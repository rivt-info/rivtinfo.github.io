*rivt* is an extensible, open source Python tool for writing and formatting
engineering documents. A rivt document, termed a *rivt doc*, is
published as a **text, HTML or PDF file** from a *rivt file*. 
A *rivt file* is a Python file (.py) that imports the 
`rivtlib <https://pypi.org/project/rivtlib/>`__ Python package and 
includes the :doc:`rivt API and markup <rvD01-markup>`. The *rivtlib* library 
is built on `docutils <https://docutils.sourceforge.io/>`__, 
`PyData Sphinx <https://pydata-sphinx-theme.readthedocs.io/en/stable/>`__
and `rst2pdf <https://rst2pdf.org/manual.html>`__.

A :term:`rivt report` is a linked compilation of multiple *rivt docs*.  
This site is an example of an HTML *rivt report*.

An interface for finding public *rivt files* on *GitHub* is  
:doc:`here <rvE04-ghsearch>`, and a collection of open source models and 
calculations, including *rivt*, can be found on 
`Google Drive <https://www.openmodels.info>`__.
