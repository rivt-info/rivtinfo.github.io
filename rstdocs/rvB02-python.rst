**B.2 Dependencies**
===================================

**[1t]** Requirements
----------------------------------------------

.. raw:: html

    <hr>

*rivt* installs Python libraries for 

- document formatting
- numerical analysis
- code simplifcation
- symbolic processing
- visualization

The minimum Python version is *3.13.0*. The recommended method for
installing *rivt* is :doc:`rivtuv <rvB01-install>`.
