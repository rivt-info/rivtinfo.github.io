2. Installation#

The following terms apply to rivt and its framework installation:

[01] Installation Terms#

rivt

rivt is a Python project. In addition to the rivtlib, it includes these packages. After rivt is installed any text editor may be used to write, edit and publish rivt docs and reports.


rivt framework

For productive writing and collaboration the rivt framework includes rivt, VSCode and VSCode extensions.


extended rivt framework

The extended rivt framework includes the following additional open source programs and services:

  1. Git and a GitHub account for version control and collaboration

  2. LaTeX for precise typesetting

  3. QCAD for diagramming

[02] Install rivt#

Method 1 - uv

This is the recommended method for most users.

  1. Install uv. The recommended method for Windows is:

winget install --id=astral-sh.uv  -e

  1. Install rivt in a subfolder by running the following shell command. This installs rivt in an isolated environment that can be deleted by deleting the rivt folder. The command file is downloaded here.

init uv rivt
cd rivt
uv add "git+https://github.com/rivtlib-dev/rivtlib.git#subdirectory=src"

Method 2 - System

Use Python installers and pip to install the packages here.

[03] Install rivt framework#

Method 1 - VSCode

  1. Install rivt

  2. Install VSCode and VSCode extensions. Extensions can be installed with this script.

Method 2 - rivt.zip

A single folder portable version of a rivt framework including Python, VSCode and all of the required libraries and extensions. VSCode and its rivt-markup extensions are installed separately to complete a rivt framework.
