B.2 | Framework#

[1] rivt frameworks#

rivt interoperates with a large number of external programs that improve its user interface and capabilities. The documented frameworks are a Basic framework and Extended framework.

[2] Basic framework#

The rivt basic framework includes programs for editing, remote storage and version control. This framework is included in portable rivt.

[3] Extended framework#

The rivt extended framework includes the rivt basic framework and additional programs and libraries for formatting, drawing, diagramming and calculating.
