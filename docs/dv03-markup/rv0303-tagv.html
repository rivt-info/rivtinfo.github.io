3.4 Tags in [V]alues#

line tags format a line of text and are denoted with _[TAG], usually at the end of the line. Block tags format a block of text that begin with _[[TAG]] on the first line and end with _[[Q]] after the last line.

KEY#

_[TAG] : line tag description

syntax : description

example

types of output

_[[TAG]] : block tag description

syntax : description

example

types of output

[01] _[#] : numbered endnote#

text _[#] text

end of text with auto endnote. _[#]

text, pdf, html

[02] _[D] : footnote text#

endnote _[D]

endnote - assigned in order _[D]

text, pdf, html

[03] _[E] : label equation#

equation label _[E]

An autonumbered equation label _[E]

text, pdf, html

[04] _[F] : label figure#

caption _[F]

An autonumbered figure label _[E]

text, pdf, html

[05] _[LS] : section link#

text _[LD] doc link

text at end of line _[LD] section number, link label

text, pdf, html

[06] _[LR] : doc link#

text _[LR] doc link

text at end of line _[LR] doc-file-name, link label

text, pdf, html

[07] _[LU] : url link#

text _[LU] url link

text at end of line _[LU] urlname, link label

text, pdf, html

[08] _[P] : new page#

_[P]

this will start a new page _[P]

text, pdf, html

[09] _[S] : sympy equation#

equation _[S]

f(x) = sin(x) + y/5 _[S]

text, pdf, html

[10] _[T] label table#

title _[T]

An autonumbered table title _[T]

text, pdf, html

[11] __ : line#

4 or more underscores ____

text, pdf, html

[12] _[[C]] : code or literal text#

_[[C]] language

_[[C]] python
print("some text")
b = 3 + 5
...
_[[Q]]

text, pdf, html

[13] _[[L]] : LaTeX#

_[[L]]

_[[L]]
\frac{\alpha}{\beta}
\sum_{n=1}^{10} n
...
_[[Q]]

pdf, html

[14] _[[T]] : topic#

_[[T]] topic

_[[T]] this a topic title : after colon will be italic
This is a topic description.
...
_[[Q]]

text, pdf, html

[15] _[[V]] : values#

_[[V]] title

_[[V]] First Floor Framing
I_1 = 100 | IN^3, CM^3, 2 | moment of intertia
L_1 = 10*1.5 | FT, M, 2| beam length
...
_[[Q]]

text, pdf, html
