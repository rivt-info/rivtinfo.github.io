3.2 Line Tags#

line tags format a line of text and are denoted with _[TAG], usually at the end of the line. Tags have a function and output scope.

KEY#

_[TAG] : description

line syntax

  • function scope

  • applicable output

_[B] : center, bold#

text _[B]

  • rv.I, rv.V

  • pdf, html

_[C] : center#

text _[C]

  • rv.I, rv.V

  • pdf, html

_[D] : footnote description#

text _[D]

rv.I, rv.V text, pdf, html

_[E] : number, label equation#

text _[C]

label _[C] rv.I, rv.V text, pdf, html

_[F] : number, label figure#

caption _[F]

rv.I, rv.V pdf, html

_[LD] : doc link#

text _[LD] doc link text

rv.I, rv.V text, pdf, html

_[LR] : report link#

text _[LR] report link text

rv.I, rv.V text, pdf, html

_[LU] : url link#

text _[LU] url link text

rv.I, rv.V text, pdf, html

_[P] : new page#

_[P]

rv.I, rv.V pdf, html

_[S] : sympy equation#

equation _[S]

rv.I, rv.V text, pdf, html

_[T] number, label table#

title _[T]

rv.I, rv.V text, pdf, html

_[V] : number, label values table#

text _[V]

rv.I, rv.V text, pdf, html

_[#] : numbered footnote#

text _[#] text

rv.I, rv.V text, pdf, html

horizontal line#

5 or more underscores _____

rv.I, rv.V text, pdf, html
