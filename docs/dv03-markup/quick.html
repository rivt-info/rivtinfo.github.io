3.5 Quick Look-up#

Scope

Read Commands

File Types

rv.V

|VALUES| rel. pth | title, [rows], -;_[V]

.csv

rv.I, rv.V

|IMG| rel. pth | caption, scale, -;_[F]

.png,.jpg

rv.I, rv.V

|IMG2| rel. pth | c1, c2, s1, s2, -;_[F]

.png,.jpg

rv.I

|TABLE| rel. pth | title, col w, l;c;r, [r], -;_[T]

.csv,.txt,.xlsx

rv.I

|TEXT| rel. pth | [[block tag]]

.txt,.tex

Notes

||DOC| rel. pth | pdf; txt; html; pdf2

pdf2 uses rst2pdf

rv.W

rv.W

||APPEND| rel. pth | num; nonum

.pdf,.txt
