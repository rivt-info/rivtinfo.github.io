**[1]** Command Line 
------------------------

.. raw:: html

    <hr>

*rivtlib* is generally run interactively in an IDE. If run from the command line
the command is

.. code-block:: bash

    python rv-dss-filename.py

where *rv-dss-* is the **rivt file number**, *d* is an alphanumeric division
label and *ss* is a two digit subdivision number respectively. 
See :doc:`here </dv-D-documents/rv-D02-folders>` for further details.
