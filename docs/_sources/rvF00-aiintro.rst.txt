**[1]** *rivt* in an AI context
------------------------------------------------------------------------ 

*rivt* runs independent of *AI*, but as software that organizes and generates
documents from a variety files, *AI* can be effective in generating content for
*rivt docs*. Since *rivtlib* is built on Python which is very well understood by
*AI* models, *rivt files* are understood by many *AI* tools..

As a context for future *rivt* development, a summary of the structure of
relevant AI models and tools is provided :ref:`here <aiback>`.  In brief, *AI*
outputs are based on a few large foundational models which provide input to many
different agents, tools and UI's that are in turn used to generate responses to
individual queries and prompts. 

The possiblity of agents that generate complete *rivt files* for specific
project requirements is currently being explored. If successful, *rivt* AI
agents are planned to be offered as a software service in the latter part of
2026.

------------------------------------


**[2]** Current AI Use Cases
----------------------------------------------------------------------- 

The *VSCode IDE* has been used to develop *rivt*. *VSCode* is integrated with
*Microsoft Copilot*, an AI coding assistant powered by the *OpenAI*
:ref:`foundation model <aiback>`. *Copilot*  recognizes *rivt markdown* and is
able to make editing and content suggestions. More importantly *Copilot* can be
used to generate tables, equations and models that can be directly
used in *rivt* documents. Examples follow.
