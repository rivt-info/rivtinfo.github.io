[2t] Center text#

Centers line of text within the page margins.

_[C]

Syntax:
    text _[C]

Example:
    This text wil be centered. _[C]

API Scope

Insert

Doc Types

text, PDF, HTML

[3t] Right justify text#

Right justifies line of text within the page margins.

_[R]

Syntax:
    text _[R]

Example:
    This text wil be right justified _[R]

API Scope

Insert

Doc Types

text, PDF, HTML

[4t] Text math#

Formats a text math expression.

_[M]

Syntax:
    text math expression  _[M]

Example:
    f(x,y) = sin(x)**2 + y/5 _[M]

API Scope

Insert

Doc Types

text, PDF, HTML

[5t] LaTeX math#

Formats a LaTeX math expression in the font specified in the style files.

_[L]

Syntax:
    LaTeX math expression  _[L]

Example:
    \frac{1}{\sqrt{x}} _[L]

API Scope

Insert

Doc Types

PDF, HTML

[6t] URL link#

_[U]

    external url, label  _[U]

Example:
    https://www.github.com, github _[U]

API Scope

Insert

Doc Types

[7t] Glossary link#

Links a term to the glossary.

_[G]

Syntax:
    text _[G] glossary term

Example:
    Links to the glossary term _[G] doc.

API Scope

Insert

Doc Types

PDF, HTML

[8t] Section link#

Creates a link to a section label defined in the API header.

_[S]
Syntax:

text _[S] section title

Example:

This creates a link to _[S] My Section Title

API Scope

Insert

Doc Types

PDF, HTML

[9t] Endnote number#

Assigns endnote number to the text in order of processing. Endnotes are defined with the block tag _[[ENDNOTE]] and are listed at the end of the doc.

_[#]

Syntax:
    text _[#] more text

Example:
    This is a sentence with an endnote _[#] tag.

API Scope

Insert

Doc Types

text, PDF, HTML

[10t] Doc link#

Links to a doc in a report. The doc title will be inserted as the link term.

_[D]

Syntax:
    text rivt_file _[D] more text

Example:
    This is a link to the rv101-filename1 _[D] doc.

API Scope

Insert

Doc Types

PDF, HTML

[11t] Variable value#

Inserts the value of var_name _[V] in the sentence.

_[V]

Syntax:
    text var_name _[V] more text

Example:
    The value of my_var is my_var _[V].

API Scope

Insert, Values

Doc Types

text, PDF, HTML

[12t] Equation number#

If the line of text is math text or a label, the text is assigned an equation number and right justified. If the line is an equation assignment the equation label is used for the label text.

_[E]

Syntax:
    math text or label _[E]

    assignment command _[E]

Example:
    3x * 4/(1+n) _[E]

    x <= 3*IN + 4.1*FT | inch, cm, 2 | example 1 _[E]

API Scope

Insert, Values

Doc Types

text, PDF, HTML
