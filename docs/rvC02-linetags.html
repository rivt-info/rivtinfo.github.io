[2t] Endnote number#

text _[#] text

Some text. _[#] Some more text.

This tag assigns an endnote number to the text in order of processing. Endnotes are defined with the block tag _[[ENDNOTE]] and are listed at the end of the doc.

[9t] Doc link#

text _[D] doc number

The phrase following the comma will be linked to the doc _[D] rA01-myfile, received yesterday.

Links to a doc in a report. The tag must be at the end of a line. Text will be continued and wrapped when formatted.

[10t] URL link#

The phrase following the comma will be linked to the specified url _[U] https://myurl.com, my link

External url link. The tag must be at the end of a line. Text will be continued and wrapped when formatted.

[11t] Equation label#

assign command or label _[E]

math text or label _[E]

equation assignment command _[E]

If the line of text is math text or a label, the text is assigned an equation number and right justified. If the line is an equation assignment the equation label is used for the label text.
