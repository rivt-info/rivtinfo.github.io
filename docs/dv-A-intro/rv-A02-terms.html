A.2 Terms#

<i>

[1] rivt Terms#

division#

A group of related rivt files organized in a report and labeled in the rivt file name. Optionally organized in a division folder.

doc#
docs#

A formatted document file output by rivt. The file is a text, PDF or HTML file.

rivt markup#

A light weight markup that wraps restructuredText and outputs reports in text, PDF and HTML system that specifies document structure and formatting.

rivt string#

Triple quoted string argument to an API function.

rivtlib#

Python package that generates docs and reports from a rivt file

[2] Python Terms#

docutils#

A Python package that processes restructured text files into HTML, LaTeX, and other formats.

namespace#

provides scope for functions and variables

restructured text#

A lightweight markup language designed to be processed by document software such as Docutils and rivt, and readable by humans.

[3] GitHub Terms#

docutils1#

A Python package that processes restructured text files into HTML, LaTeX, and other formats.

repository#

a storage location for software packages

[4] VSCode Terms#

docutils2#

A Python package that processes restructured text files into HTML, LaTeX, and other formats.
